IoT Heater

In my third master semester I attended the microcontroller applications class. The final project was an IoT Heater controlled by a RaspberryPi Zero. The main goal of this project was to develop a backup power supply to properly shut down the Pi in case of a power outage. Moreover, a dashboard for regulating and monitoring the temperature provided via a web app was part of the project. Go was used in the backend while the frontend was implemented with React.
